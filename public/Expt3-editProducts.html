About the experiment

This experiment in addition to fetching and posting data to from the server, it also sends put request to the server to update product details. The put request is sent appending the request parameters in the url. The parameters will then be extracted by the server as follows :

Code Snippet:

app.put('/shopping/updateCount/:id/:count', function (req, res) {
cart[req.params.id].count = req.params.count;
res.json(cart);
});

So when client sends PUT request to the server, the server executes the code inside the put method returning the appropriate result. Since there can be more than one put method configured for the server, the server will match the url to invoke the appropriate method. Client will send the POST request as follows :

Code Snippet:

$http.post('/shopping/addReview/'+ review+'/'+$scope.reviewId)
.success(function (response) {
console.log(response);
$scope.reviews = response;
});
}
