About the experiment

This is a very basic experiment exhibiting the use of node.js server. The server is configured with the $http.get method with all necessary logic. So when client sends GET request to the server, the server executes the code inside the get method returning the appropriate result. In this experiment, the server's get method will return a list of products to the client. Since there can be more than one get method configured for the server, the server will match the url to invoke the appropriate method. The server's get method to fetch details of a particular product given the product id will like the following:

Code Snippet:

app.get('/getDetails/:id', function (req, res) {
for(var i in products){
if(products[i].id == req.params.id){
res.json(products[i]);
}
}
});

If no product id is given, then the following get method will be invoked :

Code Snippet:

app.get('/getDetails', function (req, res) {
res.json(products);
});
